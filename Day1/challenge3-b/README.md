# Challenge3-b

Write an `Assembly (x86)` code that reads a NULL terminated string and for every
letter prints its frequency.

Obs:

If the letter doesn't apear at all, it shouldn't be printed.

Eg:

INPUT: `gigel`

OUTPUT:

    e - 1
    g - 2
    i - 1
    l - 1
