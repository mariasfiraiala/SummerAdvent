# Challenge6

Given an array with n integers and k, another integer, partition the array such
that the numbers lower than k to be the first ones and the numbers greater or
equal to k to be last.

Eg:

INPUT:
```
6 3
1 4 3 2 5 2
```

OUTPUT:

```
1 2 2 4 3 5
```
